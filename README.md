# EDC

_Every Day Context_

EDC is a tool that runs in two places:

1. On the command line i.e `e new-task`
2. As an offline-first PWA site.

This is so, as a developer I can keep track of tasks, lists or context in
general about all the things I want to get done.

Both via my phone and also while at the command line (basically anytime I am at
my computer).
